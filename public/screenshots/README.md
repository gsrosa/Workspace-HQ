# Screenshots

Place screenshots and GIFs here:

- `dashboard.png` - Dashboard screenshot
- `dashboard.gif` - Animated GIF showing key flows

## Capturing Screenshots

1. **Dashboard Screenshot**: 
   - Login to app
   - Navigate to dashboard
   - Capture full page (1920x1080)

2. **Dashboard GIF**:
   - Record screen while:
     - Creating a task
     - Editing a task
     - Scrolling virtualized list
   - Export as GIF (max 10s, optimized)
